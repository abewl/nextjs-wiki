# Simple NextJS Wiki template 

Demo [link here](https://wikidemo.codehub.com.au/)

- Static deployment
- Slug routing for nested pages
- Markdown files content parsing with [react-markdown](https://www.npmjs.com/package/react-markdown)
- Frontmatter with [gray-matter](https://www.npmjs.com/package/gray-matter)
- Sidebar nav with page tree build
- Simple search build-time page index with [Fuse.js](https://www.fusejs.io/)
- Markdown auto-styling and code highlight with [react-highlight](https://www.npmjs.com/package/react-highlight?activeTab=readme)
